# Booking Event
샐랩이라는 샐러드 판매 회사의 가상 홈페이지 애플리케이션

## 기능
- 샐랩 소개
- 샐랩 메뉴 조회
- 게시판 글 확인 및 댓글 작성
- 샐랩 주문 조회

## 사용 기술
- Front-End : React
- Back-End : Firebase
- Database : Firebase
- Deployed : Firebase

## Start
1. 레포지토리의 소스코드 pull
2. npm i
3. /src/config/MyFireBase.js 파일 작성
4. npm start
5. localhost:8080 확인
